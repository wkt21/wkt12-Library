About
-----
The WKT12 Cybersecurity Podcast explores practical cybersecurity topics through conversations, walkthroughs, and real-world examples. Each episode includes show notes, references, and related resources to help you "Build. Defend. Automate. Learn."

Episodes
--------

### Episode 001 — End Point Security (24:00)
![End Point Security cover](https://github.com/user-attachments/assets/17e23bcc-d483-473e-a466-c836074e9185)
Brief: Practical strategies for securing endpoints — covering EDR/antivirus choices, patch management, configuration hardening, and detection tuning for modern workstations and servers.

### Episode 002 — Elite Critical Thinking (42:40)
![Elite Critical Thinking cover](https://github.com/user-attachments/assets/e221b815-6871-42d3-bfe1-e9758d2146da)

### Episode 003 — Cybersecurity's Architect (39:15)
![Cybersecurity's Architect cover](https://github.com/user-attachments/assets/a5a5aa9e-8c01-4c15-a0ac-ab68e44ba6b5)

### Episode 004 — The Art in Vulnerability (22:20)

### Episode 005 — WKT12 Cyber Operations Manual (01:01:59)
