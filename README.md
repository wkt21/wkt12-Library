# WKT12

A curated public site for cybersecurity work, OSINT content, Kindle books, and Spotify podcast displays.

## 🌐 Live Site

- **Website**: [wkt12.tech](https://wkt12.tech)

## 📚 Sections

- **Home** - Welcome and introduction
- **Kindle Books** - Curated collection of cybersecurity and OSINT reading materials
- **Spotify Podcast** - Featured security and tech podcasts
- **About** - Learn more about the project and creator

## 🔍 Features

- **Cybersecurity Resources** - In-depth articles and guides on security topics
- **OSINT Tools & Techniques** - Open Source Intelligence resources
- **Book Recommendations** - Handpicked Kindle books for security professionals
- **Podcast Aggregation** - Curated Spotify podcast displays
- **Static Site Performance** - Fast, secure, and reliable hosting

## 📋 Purpose

This repository powers a static Jekyll site hosted on GitHub Pages and mapped to the `wkt12.tech` domain. It serves as a central hub for cybersecurity enthusiasts, OSINT practitioners, and tech professionals seeking curated content and resources.

## 🛠️ Built With

- **Jekyll** - Static site generator
- **GitHub Pages** - Hosting platform
- **Markdown** - Content creation
